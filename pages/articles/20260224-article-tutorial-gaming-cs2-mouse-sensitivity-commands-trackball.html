# 20260223-recordings-cs2-mouse-sensitivity-higher-for-trackballs ## Intro Hello, ladies and gentlemen. ## Hook Have you ever noticed that in _Counter-Strike 2_, the maximum sensitivity you can set through the in-game menu feels pretty limited compared to other FPS games? ## Testimony I know I have. I've played CS for more than 10 years now, and for a decent chunk of that time until now, I have personally used a trackball mouse: a fingerball called the _ELECOM Deft Pro_, for all of my games. However, when _Counter Strike 2_ came out, I initially had a lot of trouble because you can only turn up the sensitivity to this "eight" value, which isn't nearly as high as which I normally play games, like _Left 4 Dead 2_, which has _Counter-Strike_ DNA all over it. ## How to ### Enable the console To enable the command line interface console in Counter-Strike 2 (CS2), follow these steps: 1. Launch CS2 and go to the main menu. 2. Click the gear icon (⚙️) in the top-left corner to open the Settings menu. 3. Navigate to the Game tab. 4. Find the option labeled "Enable Developer Console (~)". 5. Change the setting from "No" to "Yes". Once enabled, you can open the console by pressing the ~ (tilde) key on your keyboard. This key is typically located just below the Esc key, to the left of the 1 key. ### The commands The `m_pitch` program controls vertical (up/down) mouse sensitivity, and the `m_yaw` controls horizontal (left/right) mouse sensitivity. These client-side console commands allow fine-tuning of mouse responsiveness independent of in-game sensitivity. One thing that I've read up on the commands is that, supposedly, the default value for both is `0.022`. In practice, however, the rough equivalent values for each is wildly different in regards to achieving roughly the same sensitivity on both axes. #### Sytax For both programs, you put the program name, a space, and a real number (i.e., any number over and number zero, with a decimal being optional). ##### Suggested My recommendation, especially for trackball users on Linux distributions with XLibre display system defaults, I'd recommend the following values: - `m_yaw 0.019` And... - `m_pitch 0.019` That should be a good starting place. ## Conclusion Have fun!
